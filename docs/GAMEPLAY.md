# 游戏玩法说明文档

## 游戏概述

《启蒙》是一款Roguelike卡牌构筑游戏，玩家将扮演一名英雄，通过不断战斗和收集卡牌，攀登高塔，最终击败Boss获得胜利。

## 核心玩法

### 1. 游戏目标
- 选择一名职业，组建卡组
- 攀登3层塔楼，击败各层敌人
- 战胜最终Boss，完成通关

### 2. 游戏流程

```
主菜单 → 职业选择 → 地图探索 → 战斗 → 奖励 → 下一层 → Boss战 → 通关
```

#### 2.1 主菜单
- **开始游戏**: 进入职业选择界面
- **PVP对战**: 功能开发中
- **帮助**: 查看游戏基本说明

#### 2.2 职业选择
游戏提供3个职业，每个职业有不同的玩法风格：

| 职业 | 生命值 | 特点 | 解锁条件 |
|------|--------|------|----------|
| **战士** | 80 | 高防御、稳定输出 | 初始解锁 |
| **刺客** | 65 | 高机动、过牌能力强 | 战士通关1次 |
| **法师** | 60 | 低生命、高爆发 | 战士通关1次 |

#### 2.3 地图探索
每层地图包含以下节点类型：

| 节点类型 | 图标 | 说明 |
|----------|------|------|
| **普通战斗** | ⚔️ | 遭遇普通敌人，获得金币奖励 |
| **精英战斗** | 👹 | 遭遇强力敌人，获得更好奖励 |
| **休息点** | 🔥 | 恢复30%最大生命值 |
| **商店** | 🏪 | 购买卡牌和遗物（Demo版暂未开放） |
| **Boss战** | 👑 | 每层最终战斗，必须击败才能进入下一层 |

### 3. 战斗系统

#### 3.1 基础规则
- **回合制**: 玩家回合 → 敌人回合 → 循环
- **能量系统**: 每回合3点能量，打出卡牌消耗能量
- **手牌**: 每回合开始时抽取5张牌
- **牌堆**: 抽牌堆空时，弃牌堆洗牌进入抽牌堆

#### 3.4 战斗机制详解

##### 格挡机制
- 技能卡可以获得格挡值
- 受到攻击时，先扣减格挡值，再扣减生命值
- 回合结束时，格挡值减半（除非有保留格挡的能力）
- 部分敌人有"破甲"攻击，可以无视格挡

##### 伤害计算
```
实际伤害 = 基础伤害 + 力量加成 - 敌人格挡
```

##### 狂战士之血效果
```
伤害加成 = (1 - 当前HP/最大HP) × 10 × 能力等级
```

### 7. 游戏技巧

#### 7.1 通用技巧
1. **合理规划能量**: 不要浪费能量，尽量每回合用完
2. **保留关键牌**: 某些牌留到关键时刻使用效果更好
3. **注意敌人意图**: 根据敌人意图决定是攻击还是防御
4. **格挡管理**: 预估敌人伤害，不要过度防御

#### 7.2 职业技巧

##### 战士
- 利用塔盾的每回合格挡，减少防御卡的使用
- 盾击是攻防一体的优秀卡牌
- 钢铁意志可以让格挡保留，适合长线战斗

##### 刺客
- 利用过牌能力快速找到关键牌
- 暗影匕首配合低费攻击卡输出更高
- 准备卡可以0费过牌，非常灵活

##### 法师
- 生命值较低，注意保持格挡
- 元素法典提供额外过牌，利用抽牌优势
- 能力卡较多，尽早打出能力建立优势

## 常见问题

### Q: 如何解锁新职业？
A: 使用战士通关一次即可解锁刺客和法师。

### Q: 为什么有些卡牌打不出去？
A: 请检查能量是否足够，以及是否选择了正确的目标。

### Q: 战斗失败了怎么办？
A: 游戏结束，需要重新开始。Roguelike游戏的特点是每次游戏都是全新的体验。

### Q: 如何恢复生命值？
A: 在地图上的休息点（🔥）可以恢复30%最大生命值。

### Q: 敌人的"破甲"是什么意思？
A: 破甲攻击可以无视你的格挡，直接造成伤害。
