# 数值设计文档

## 1. 核心数值概览

### 1.1 数值设计原则
- **简洁性**: 数值简单易懂，便于玩家快速上手
- **平衡性**: 三个职业各有特色，无明显强弱之分
- **成长性**: 通过卡牌和能力获得成长感
- **策略性**: 不同卡牌组合产生不同效果

### 1.2 数值范围

| 数值类型 | 最小值 | 典型值 | 最大值 |
|----------|--------|--------|--------|
| 生命值 | 60 | 65-80 | 100 |
| 能量 | 3 | 3 | 4（临时） |
| 卡牌费用 | 0 | 1-2 | 3 |
| 伤害 | 4 | 6-14 | 18（敌人） |
| 格挡 | 3 | 5-12 | 15 |
| 抽牌 | 1 | 2 | 5（初始） |

## 2. 职业数值设计

### 2.1 职业属性对比

| 属性 | 战士 | 刺客 | 法师 |
|------|------|------|------|
| 最大HP | 80 | 65 | 60 |
| 初始卡组数量 | 10 | 9 | 9 |
| 初始攻击卡 | 6 | 4 | 3 |
| 初始防御卡 | 4 | 3 | 2 |
| 初始技能卡 | 0 | 2 | 4 |
| 特色 | 高防稳定 | 过牌机动 | 低血高爆 |

### 2.2 职业定位分析

#### 战士 (Warrior)
```
定位: 坦克/稳定输出
优势: 
  - 最高生命值(80)
  - 塔盾提供稳定防御
  - 攻防兼备的卡牌
劣势:
  - 过牌能力弱
  - 爆发力不足

设计意图: 适合新手，容错率高，玩法稳健
```

#### 刺客 (Assassin)
```
定位: 机动/连击
优势:
  - 过牌能力强(2张抽牌卡)
  - 暗影匕首增伤
  - 0费卡牌灵活
劣势:
  - 生命值中等(65)
  - 防御能力一般

设计意图: 适合喜欢快节奏、多操作的玩家
```

#### 法师 (Mage)
```
定位: 爆发/控场
优势:
  - 元素法典每回合抽牌
  - 能力卡丰富
  - 0费准备卡
劣势:
  - 最低生命值(60)
  - 初始攻击卡少

设计意图: 适合高手，需要精确计算和规划
```

### 2.3 职业平衡性分析

```
有效生命值 = 最大HP + (平均每回合格挡 × 预计战斗回合数)

战士: 80 + (4 × 10) = 120
刺客: 65 + (0 × 10) = 65  [需要主动防御]
法师: 60 + (0 × 10) = 60  [需要主动防御]

结论: 战士生存能力最强，刺客和法师需要通过卡牌弥补防御
```

### 5.2 遗物价值评估

```
塔盾 (战士):
  10回合战斗 = 40格挡 ≈ 8张防御卡
  价值: 非常高，大幅降低防御压力

暗影匕首 (刺客):
  10回合战斗 = 30额外伤害
  价值: 中等，需要配合低费攻击卡

元素法典 (法师):
  10回合战斗 = 10张额外卡牌
  价值: 高，提升卡组流畅度
```

## 7. 战斗评价系统

### 7.1 评价标准

| 评价 | 受伤阈值 | 剩余HP比例 |
|------|----------|------------|
| 完美胜利 | 0 | 100% |
| 精彩胜利 | ≤10 | >87.5% |
| 胜利 | ≤20 | >75% |
| 险胜 | >20 | >30% |
| 惨胜 | >20 | ≤30% |

### 7.2 评价计算

```typescript
const damageTaken = maxHp - currentHp;

if (damageTaken === 0) return '完美胜利';
if (damageTaken <= 10) return '精彩胜利';
if (damageTaken <= 20) return '胜利';
if (currentHp / maxHp > 0.3) return '险胜';
return '惨胜';
```

## 9. 数值调整建议

### 9.1 当前版本问题

1. **法师偏弱**: 生命值过低，容错率低
2. **部分卡牌出场率低**: 如旋风斩、不屈意志
3. **精英敌人单一**: 只有重装骑士一种

### 9.2 优化建议

| 问题 | 建议调整 |
|------|----------|
| 法师生存 | 法师HP 60→65，或元素法典额外+1初始能量 |
| 旋风斩 | 伤害 6→8，或费用 2→1 |
| 不屈意志 | 触发条件 HP<50%→HP<60%，或费用 2→1 |
| 精英敌人 | 增加1-2种精英敌人，丰富战斗体验 |

## 10. 扩展数值预留

### 10.1 未来扩展方向

1. **更多职业**: 牧师（治疗）、猎人（召唤）等
2. **更多卡牌**: 每张卡3个升级版本
3. **遗物系统**: 20+个可收集遗物
4. **事件系统**: 随机事件，影响数值

### 10.2 数值增长曲线

```
Demo版: 3层，20张卡
完整版建议: 5层，50+张卡，30+遗物

每层难度增长: 敌人HP +20%，伤害 +15%
卡牌成长: 升级后效果 +50%
```
